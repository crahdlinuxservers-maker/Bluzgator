# 👺 Bluzgator v4.2 Armagedon
**Zaawansowany, inteligentny i wyjątkowo złośliwy bot rozrywkowy na platformę Discord.**

✨ Funkcje
*   **Gigantyczny Słownik:** Baza ponad 130 unikalnych przymiotników, rzeczowników i dodatków generująca setki tysięcy kombinacji pocisków.
*   **System Lojalnościowy:** Bot rozpoznaje swojego stwórcę (**Crahd**), oferując mu ochronę przed zniewagami oraz specjalne przywileje.
*   **Tryb Snajpera:** Precyzyjne narzędzie nękające – bot automatycznie odpowiada na każdą wiadomość wskazanego celu.
*   **Moduł Analizy:** Złośliwy wykrywacz "dzbanizmu" (0-100%) oraz generatory zjadliwych horoskopów na każdy dzień.
*   **System Rankingowy:** Dynamiczna lista TOP 5 największych ofiar bota (Ranking Dzbanów) prowadzona w czasie rzeczywistym.
*   **Modern UI (Embeds):** Elegancki interfejs oparty na Rich Embeds Discorda, zapewniający czytelność nawet przy dużej ilości funkcji.

🛠️ Technologia
*   **Python 3.13+**
*   **Discord.py** (Asynchroniczna biblioteka do obsługi API Discorda)
*   **Python-dotenv** (Bezpieczne zarządzanie danymi wrażliwymi)
*   **Random & Asyncio** (Logika generowania treści i wielozadaniowość)

👑 Przywileje Pana (Tylko dla Crahda)
*   `!snajper @użytkownik` – Aktywacja/dezaktywacja automatycznego nękania celu.
*   `!czysc [ilość]` – Szybkie sprzątanie wiadomości bota i komend z kanału tekstowego.
*   `!poklon` – Deklaracja absolutnego poddaństwa bota wobec stwórcy.
*   `!changelog` – Podgląd najnowszych modyfikacji wprowadzonych w wersji Armagedon.

🚀 Instalacja
1. Pobierz repozytorium na dysk lokalny.
2. Zainstaluj wymagane zależności: `pip install discord.py`.
3. Włącz **Message Content Intent** w panelu deweloperskim Discorda.
4. Wstaw swój unikalny Token bota w sekcji konfiguracji w pliku `main.py`.
5. Uruchom aplikację: `python main.py`.
